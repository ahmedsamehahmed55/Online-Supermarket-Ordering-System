# Online SuperMarket Ordering System

A C++ project simulating an online ordering system with inventory management using custom data structures.

## Features
- Add, search, and remove items
- Shopping cart functionality
- Merge sort for inventory
- Object-oriented design

## Folder Structure
- `src/` → C++ source files
- `include/` → Header files
- `data/` → Text/data files

## How to Compile
```bash
g++ src/*.cpp -Iinclude -o OnlineSuperMarket
./OnlineSuperMarket
